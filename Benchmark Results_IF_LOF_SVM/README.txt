bnlearn Package installation:
-----------------------------
bnlearn is available on CRAN (https://cran.r-project.org) and can be downloaded from its web page (https://cran.r-project.org/web/packages/bnlearn/index.html)

It can be installed with a simple: 

install.packages("bnlearn")
or
install.packages(​"http://www.bnlearn.com/releases/​bnlearn_latest.tar.gz")
